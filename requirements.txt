# - macOS Apple Silicon: `pip install -r requirements.txt` installs a regular
#   PyTorch build that can use the Apple GPU through the `mps` device.
# - Linux + NVIDIA CUDA: install the matching CUDA wheel from
#   https://pytorch.org/get-started/locally/ first, then install this file.
#   The torch range below is intentionally compatible with Linux CUDA wheels
#   such as cu128 as well as the default macOS wheel.
